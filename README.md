<h1 align="center">🎯 JD Match Assistant</h1>
<p align="center"><em>别再一个个 JD 看完不知道投不投了。<br>把你的画像配上去，AI 帮你过滤、打分、排名、出方案。</em></p>

---

## 📊 实际战绩

```
扫了字节跳动 ByteIntern 16 个 JD
    ↓ 硬门槛过滤毙掉 8 个（研发岗 / 论文 / 专业硬限）
    ↓ 六维雷达图分析 8 个
    ↓ 实时排名收敛
    ↓
最终 5 个精准匹配 → 输出完整投递方案
```

| 指标 | 数据 |
|------|:--:|
| 扫描 JD | 16 |
| 毙掉 | 11（69%） |
| 入围 | 5 |
| 第一志愿匹配度 | ⭐⭐⭐⭐⭐ |
| 耗时 | ~2 小时 |

## 🚀 30 秒上手

### 方式一：已有画像（30 秒）

直接把你的画像贴给 Claude，然后发 JD：

```
学历：XX大学 XX专业
核心技能：① PE ② Agent ③ ...
独有优势：① ... ② ...
硬伤：① ... ② ...
意向：AI 数据运营
```

### 方式二：还没画像（3 分钟）

发 `prompt.md` 给 Claude，回答 5 个问题自动生成画像：

> 1. 你学什么的？
> 2. 你最骄傲的一个项目？
> 3. 别人夸你最多的一句话？
> 4. 你最不想做什么岗位？
> 5. 你在怕什么？

## 🔗 搭配使用：数字分身采访官

这两个工具是一对——一个向内挖，一个向外投：


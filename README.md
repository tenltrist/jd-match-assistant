# JD Match Assistant

> 把你的 JD 扫盲变成精准配对——AI 帮你把"这个岗位适合我吗"变成一个可复用的工作流。

## 这是什么

一个 Claude Prompt。你发 JD，它帮你：
- **硬门槛过滤** — 能不能投，不能直接毙
- **六维雷达图** — 专业/技能/优势/竞争/展示/时长 六个维度打分
- **实时排行榜** — 每进一个 JD，刷新当前 TOP 3
- **投递方案** — 喊停时输出完整方案 + 简历推荐语 + Cover Letter

扫了 16 个字节跳动 ByteIntern JD，用这个流程筛出 5 个精准匹配——毙掉率 69%。

## 怎么用

1. 复制 `prompt.md` 的内容发给 Claude
2. 回答 5 个入门问题（或直接贴你的画像）
3. 一条条发 JD
4. 看着排行榜往前进
5. 喊停，拿投递方案

## 和「数字分身采访官」配合使用

这两个工具是一对——一个向内，一个向外：

1. 先用**数字分身采访官**做一次 11 阶段深度自我采访，搞清楚你是谁
2. 再用**JD Match Assistant**拿画像去扫 JD，搞清楚你适合去哪
